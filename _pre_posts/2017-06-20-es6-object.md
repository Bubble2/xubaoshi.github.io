<h1>ECMAScript.6 对象相关</h1>
<h2>对象字面量</h2>

    function createPerson(name,age){
        return {
            name:name,
            age:age
        }
    }

上述代码 createPerson函数返回一个属性名与当前函数参数名相同的对象，如果使用es6语法书写起来会更加简单。

    function createPerson(name,age){
        return {
            name,
            age
        }
    }
<h2>简化对象方法</h2>

    // es5
    var person = {
        name:"Nicholas",
        sayName:function(){
            console.log(this.name);
        }
    }

    // es6
    var person = {
        name:'Nicholas',
        sayName(){
            console.log(this.name);
        }
    }
